BAUHAUS (1919–1933)

Founded in Germany in 1919 by Walter Gropius.

Core ideas:
- Form follows function
- Art + technology unity
- Minimalism
- Functional design
- No ornamentation

Key vocabulary:
- Vorkurs
- geometric abstraction
- primary colors
- functionalism
- sans-serif typography

Important facts:
Closed by Nazis in 1933.
Influenced modern architecture and graphic design.

Possible sources:
MoMA Bauhaus Collection
Bauhaus Archive



SWISS STYLE (1950s–1970s)

Also called International Typographic Style.

Core ideas:
- Grid-based layouts
- Clean typography
- Objective design
- Mathematical structure
- Asymmetry

Key vocabulary:
- grid system
- sans-serif typography
- Helvetica
- asymmetrical layout
- objective design

Important facts:
Developed in Switzerland in 1950s.
Focused on readability and clarity.

Possible sources:
Design Museum
MoMA Typography



NEW WAVE TYPOGRAPHY (1970s–1980s)

Reaction against Swiss Style.

Core ideas:
- Broke grid rules
- Experimental layouts
- Expressive typography
- Layering
- Distortion

Key vocabulary:
- expressive typography
- layering
- grid disruption
- visual texture
- experimental layout

Important facts:
Wolfgang Weingart helped pioneer New Wave.
Started in Switzerland and Germany.

Possible sources:
MoMA Graphic Design
Cooper Hewitt Museum
