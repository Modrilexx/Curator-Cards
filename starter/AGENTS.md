AI Instructions for Curator Cards

Tone Requirements:
- Use museum-label style writing
- Clear and factual language
- No opinions or storytelling
- Professional tone

Length Requirements:
- Each curator card must be 80–120 words

Vocabulary Requirements:
Use real design vocabulary such as:

Bauhaus:
- Vorkurs
- geometric abstraction
- functionalism
- primary colors
- sans-serif typography

Swiss Style:
- grid system
- asymmetrical layout
- Helvetica
- objective design
- mathematical grid

New Wave Typography:
- expressive typography
- layering
- grid disruption
- experimental layout
- visual texture

Rules:

Do NOT use vague words like:
- cool
- nice
- modern
- interesting

Always include:
- Historical dates
- Key principles
- Design vocabulary
